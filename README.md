# Lista-de-Tareas
Descripción del proyecto

Este proyecto es una aplicación web sencilla que permite registrar tareas académicas mediante un formulario. Cada tarea incluye la materia, la fecha de entrega y un estado representado por un checkbox. Al enviar el formulario, las tareas se agregan dinámicamente a una tabla, facilitando su visualización y control. El proyecto está desarrollado utilizando únicamente HTML, CSS y JavaScript, sin el uso de frameworks o librerías externas.

Instrucciones de uso

Abrir el archivo index.html en un navegador web.

Completar el formulario ingresando la materia y la fecha de la tarea.

Presionar el botón de registro.

La tarea se mostrará automáticamente en la tabla.

Utilizar el checkbox para marcar la tarea como completada.

Tecnologías utilizadas

HTML5 para la estructura del formulario y la tabla.

CSS3 para el diseño y estilos visuales.

JavaScript para la lógica de funcionamiento y manipulación del DOM.
